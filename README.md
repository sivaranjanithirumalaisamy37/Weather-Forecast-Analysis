# Weather Forecast Analysis Using Fundamental Data Science Experiments

This project applies the first ten fundamental data science lab concepts to a weather forecast dataset using Python. It includes data preprocessing, correlation analysis, logistic regression, linear regression, random sampling, Z-test, NumPy operations, data cleaning, weather simulation, and trend analysis.

## Project Description
The main aim of this project is to analyze weather data and extract meaningful insights using basic data science techniques. The dataset contains attributes such as Temperature, Humidity, Wind Speed, Cloud Cover, Pressure, and Rain. Using these features, the project performs both statistical and machine learning based analysis.

## Experiments Covered
1. Correlation Matrix  
2. Logistic Regression for Rain Prediction  
3. Linear Regression for Temperature Prediction  
4. Random Sampling  
5. Z-Test  
6. NumPy Array Operations  
7. Handling Missing and Invalid Data  
8. Weather Simulation  
9. Trend Analysis  
10. Overall Weather Data Analysis

## Conclusion
This project demonstrates how fundamental data science experiments can be applied to a real-world weather dataset. It helps in understanding data preprocessing, statistical testing, machine learning models, and visualization techniques in a simple and practical way.
